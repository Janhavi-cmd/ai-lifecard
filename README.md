# ⚡ AI LifeCard — Unified Personal Intelligence Dashboard

> **Full-stack AI intelligence platform analyzing financial, career & burnout patterns across 500+ users. Built dual ML engine — RandomForest for pattern classification and IsolationForest for anomaly detection — with a context-aware chatbot assistant. Deployed on Railway + Vercel with Firebase auth.**
